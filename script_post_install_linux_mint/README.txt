#*****************************README**************************************************************************
Actualización y instalación de programas en un sistema operativo Linux Mint.
Con este script se instalaran los programas básicos para trabajar.
También se realizaran configuraciones básicas pero necesarias para un buen funcionamiento del sistema. 
A continuación se detalla una lista de los programas que se van a instalar y su función principal:

BUILD-ESSENTIAL //herramientas necesarias para compilación de software
VLC // reproductor de video
LIBREOFFICE // Suite de offimatica
DOSBOX //emulador de MS-DOS
VIRTUALBOX // virtualizador de sistemas operativos
CMUS //reproductor de música desde consola
DEFAULT-JRE //JRE para ejecutar programas creados en Java
DEFAULT-JDK //JDK para programar en Java
STEAM //plataforma online para comprar videojuegos online
UNRAR // para descomprimir archivos
XTERM // para usar code::blocks
VIM // editor de texto plano desde consola
GPARTED // particionador de discos
SYNAPTIC // Gestor de paquetes modo gráfico
MAME // emulador de maquinas recreativas
HTOP // igual al clásico top pero con más funciones
CODEBLOCKS // IDE para C/C++
GAMBAS3 // Lenguaje de programación Gambas3
PYTHON3 // Lenguaje de programación Python 3
CHROMIUM // Navegador web Chrome en su versión libre
GMUSICBROWSER // Reproductor de música
GIMP // Programa para diseño gráfico
OPENSSH-SERVER //conexiones remotas usando SSH
GIT // Programa de control de versiones
KAZAM //Programa para grabar videos
ARDUINO IDE // Programa para programar placas Arduinos
FRITZIG // Editor visual para Arduino
SCRATCH // Lenguaje de programación Scratch
CHEESE // Programa para tomar fotografias con la camara web
KODI // Plataforma multimedia para reproducir videos, música, etc
LIFEREA //Lector de noticias RSS
SOUNDCONVERTER // convertir archivos de video a audio en formato .ogg o .mp3
WGET //descarga de contenidos desde servidores web de forma simple por terminal
COMIX // Lector de comics
INKSCAPE // Programa para diseño vectorial
SQLITE3 // Sistema gestor de bases de datos sqlite versión 3
SQLITEMAN // Gestor gráfico para SQLite 3
MYSQL-SERVER // MySQL versión servidor
MYSQL-Workbench // Gestor gráfico para MySQL
SPOTIFY // Red social que permite escuchar canciónes de manera online
